周一刷推的时候，看到了这样的一个用来统计敲代码时间的应用，之前也找到过别的应用，比如 [timing](https://timingapp.com/?lang=en) ，好用但是价格有点高，一个月需要 8 欧，那就没有必要了。找到了平替美滋滋。

应用名叫 [wakatime](https://wakatime.com/dashboard)，免费版可以保存两周的记录这样也算够用了，通过给开发工具安装插件，就可以统计工具使用的时长。

![image.png](https://obsidian-picgo-le.oss-cn-hangzhou.aliyuncs.com/img/20230609161344.png)

另一个则是在 iOS 上开通了 ChatGPT-Plus，并且一次就在 openai 上成功绑定信用卡，运气也是非常的爆棚。记得在绑定之前研究了两个小时才下手的，也不枉我花费这么长时间。

![image.png](https://obsidian-picgo-le.oss-cn-hangzhou.aliyuncs.com/img/20230609162518.png)

后续就可以继续使用 API key 了，正在重新写 ChatGPT 相关的应用。

最后就是这周的代码时长了 👀
